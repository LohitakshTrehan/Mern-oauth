# Mern-oauth
Create a config folder and inside it create a keys.js file.<br>
```
module.exports = {
    googleClientID: 'googleCLientID' ,
    googleClientSecret:'googleClientSecret',
    facebookClientID:'fbclientID',
    facebookClientSecret:'fbclientSecret',
    mongoURI:'mlabURI'
}
```
## To run
npm install<br>
cd client<br>
npm install<br>
cd ..<br>
npm run dev
